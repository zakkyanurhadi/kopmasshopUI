# Blue Shop HTML Template (UI)

Template statis dari komponen frontend `src/components` dan `src/views/app`.

## Entry Point
- `index.html`

## Halaman
- `index.html` -> Home
- `halaman1.html` -> Browse Category
- `halaman2.html` -> Product Detail
- `halaman3.html` -> Store Detail
- `halaman4.html` -> Cart
- `halaman5.html` -> Checkout

## Catatan
- Semua halaman menggunakan Tailwind CDN:
  - `<script src="https://cdn.tailwindcss.com"></script>`
- Interaksi dasar halaman ada di `nav.js`.
- Assets gambar ada di `assets/images`.
